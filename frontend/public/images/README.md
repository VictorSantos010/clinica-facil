# Pasta de Imagens

Esta pasta contém imagens estáticas do projeto.

## Estrutura

- `banners/` - Banners e imagens hero para a home page
- Outras imagens podem ser adicionadas aqui

## Como usar

Para usar imagens desta pasta no código, use o caminho relativo a partir de `public`:

```jsx
<img src="/images/banners/banner-home.jpg" alt="Banner" />
```

ou

```css
background-image: url('/images/banners/banner-home.jpg');
```

## Tamanhos recomendados

- **Banner Home**: 1920x600px (ou proporção similar)
- **Formato**: JPG, PNG ou WebP
- **Peso**: Máximo 500KB para melhor performance
